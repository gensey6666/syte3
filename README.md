# Самопомощь и мотивация

Небольшая веб‑платформа психологической самопомощи для ежедневной поддержки, отслеживания состояния и ведения дневника.

## Как запустить

- Откройте файл `selfhelp-platform/index.html` в браузере (Chrome / Edge / Firefox).

Если браузер блокирует работу `dialog` или скачивание экспортируемого файла, попробуйте другой современный браузер.

### Запуск через Python (если установлен)

В PowerShell:

```powershell
cd C:\Users\rrors\selfhelp-platform
python -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## Что внутри

- **Настроение**: отметка 1–10, теги, заметка, мини‑график 7 дней
- **Дневник**: записи + поиск
- **Упражнения**: короткие техники (дыхание, заземление, мысль→проверка, ценности)
- **Упражнения**: расширенный набор быстрых практик (10 шт.)
- **Аккаунты**: регистрация, вход и выход
- **Статьи**: встроенные материалы для чтения внутри платформы
- **Цели**: цель + ценность + следующий шаг (≤15 минут), отметка выполнения
- **Настройки**: экспорт/импорт JSON, полное удаление данных

## Важно

Инструмент не заменяет профессиональную помощь. Если вам небезопасно — обратитесь к близким и в экстренные службы вашего региона.
